# Multimodal DeepFake Detection System

## Overview
This project detects deepfake content using:
- Spatial features (images)
- Temporal features (video)
- Audio signals
- Frequency domain analysis

## Features
- Multimodal fusion (attention-based)
- Real-time inference ready
- API + UI support
- Explainability (Grad-CAM)
